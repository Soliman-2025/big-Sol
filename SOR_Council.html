✅ هذا النظام لا يخزن بيانات شخصية ولا يقوم بتتبع الأفراد، ويعمل كأداة تحليل مؤقتة للبيانات المدخلة فقط.

🏛 تعريف النظام

واجهة تشغيل (Web Interface) لنظام تحليل بيانات داخلي متوافق مع متطلبات الحوكمة الوطنية. يعمل النظام كأداة تحليلية استشارية تُقدّم مخرجات للمساعدة في اتخاذ القرار فقط.

🔐 الأمن السيبراني — NCA Saudi

• الاتصال مشفّر عبر HTTPS بالكامل

• لا تُخزَّن بيانات المستخدمين على خوادم خارجية

• جميع العمليات تتم في الذاكرة المؤقتة (Stateless Processing)

• الجلسة تنتهي تلقائياً عند إغلاق المتصفح

📋 حماية البيانات — PDPL

• لا تُجمَع بيانات شخصية من المستخدمين

• البيانات المُدخَلة تُعالَج مؤقتاً ولا تُحفَظ

• التحليلات تُنتَج بواسطة نماذج ذكاء اصطناعي وتُعرَض للمستخدم مباشرة

• لا يوجد تتبع للأفراد أو جمع لبيانات التصفح

🏢 حوكمة البيانات — سدايا

• مصدر البيانات: مُدخَلة من المستخدم مباشرة

• تصنيف البيانات: بيانات مؤسسية مؤقتة — غير محفوظة

• ملكية البيانات: الجهة المستخدمة — لا تنتقل لأي طرف ثالث

• معالجة الذكاء الاصطناعي: عبر Anthropic API وفق سياسة الخصوصية الخاصة بهم

⚖️ إخلاء المسؤولية

التوصيات الصادرة عن هذا النظام استشارية بحتة ولا تُشكّل قراراً ملزماً. المسؤولية البشرية مرجع نهائي دائماً. القرار الأخير لصاحب الصلاحية وحده.
